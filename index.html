# Ritun OS v2.0

Welcome to my interactive portfolio! This project simulates a full desktop environment in the browser to showcase my skills in Full Stack Architecture.

## 🛠️ Tech Stack

Backend

  • Django (Python)
  • REST APIs
  • PostgreSQL

Frontend

  • Tailwind CSS
  • Alpine.js
  • HTML5 Canvas

## ✨ Key Features

  • 🖥️ Window Management: Draggable, minimizable, maximizable windows.
  • 🚀 Boot Sequence: Realistic OS startup.
  • 🎨 Glassmorphism: Modern UI design.
  • Terminal Commands: Custom Terminal Commands (type help)
